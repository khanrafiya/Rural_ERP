## 📖 Overview

Rural ERP is a school management platform designed specifically for government and rural schools, where internet connectivity is often limited. It provides a clean, fast, and low-bandwidth-optimized interface for managing student records, academics, and administrative workflows — with multilingual support to serve diverse communities.

---

## ✨ Features

- 🎓 **Student Management** — Add, edit, view, and delete student records with ease
- 🔍 **Smart Search & Filtering** — Search by name, ID, roll number, or class
- 🔐 **Role-Based Access Control** — Admin, Principal, and Teacher roles with permission-based editing
- 📋 **Detailed Student Profiles**
  - Personal: Name, Date of Birth, Gender, Category
  - Academic: Class, Roll Number, Enrollment Status
  - Family & Contact: Parent details, phone, address, village/city
- 📊 **Dashboard Overview** — Total students, attendance rate, daily presence, new admissions
- 📱 **Progressive Web App (PWA)** — Installable on mobile devices for an app-like experience
- 🔔 **Push Notifications** — Powered by Firebase Cloud Messaging
- 🌐 **Multilingual Ready** — Built to support diverse regional languages
- ⚡ **Low-Bandwidth Optimized** — Fast performance even on slow networks
